# AnkiBridge settings

- **port** — TCP port the local server listens on. Default `48731`.
- **bind_address** — Interface to bind. `0.0.0.0` allows other devices on your
  Wi-Fi (your phone) to connect. Set to `127.0.0.1` to allow this computer only.
- **log_to_file** — Write logs to `user_files/ankibridge.log` inside the add-on
  folder.
- **enable_mdns** — Reserved for future Bonjour/mDNS auto-discovery. No effect
  in the MVP.

Restart Anki (or use *Restart server* in the status window) after changing
these.
